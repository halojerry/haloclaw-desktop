## 建议

1. **环境配置完善**：建议添加 `.env.example` 文件，包含以下环境变量示例：
   - `DASHSCOPE_API_KEY`：通义千问API Key
   - `JWT_SECRET`：JWT密钥
   - `H2_CONSOLE_ENABLED`：H2控制台开关
   - `CLOUD_API_URL`：云端API地址
   - `NEW_API_KEY`：New API密钥

2. **启动前准备**：
   - 确保Java 21环境已安装
   - 配置有效的 `DASHSCOPE_API_KEY`
   - 如需数据库持久化，可切换到MySQL profile

3. **首次启动建议**：
   ```bash
   # 设置环境变量
   export DASHSCOPE_API_KEY=your-api-key
   
   # 开发模式启动
   cd mateclaw-server
   mvn spring-boot:run
   ```
